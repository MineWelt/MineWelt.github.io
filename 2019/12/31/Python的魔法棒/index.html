还记的第一次见到 “init” 的心情吗？实在不知道这个拥有双下横线的到底是什么东西.

What’s the python’s magic methods ?

魔法方法是python的内置方法，几乎每一个魔法方法都有一个对应的内置函数，或者运算符,它是面向对象的一切.

init

“init“ 是什么呢？其实它就是一个构造器， 当你创建一个类的实例的时候， 调用 “init“ 是为了对这个实例进行初始化赋值.就像是类的固有属性.
