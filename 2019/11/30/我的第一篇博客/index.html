我的第一篇博客

写在前面

其实之前从来没有过建立自己私人博客的想法，但是有一个朋友搞了起来，发现挺有意思的，那就当玩玩吧。抱着这个心态，我的第一个自己搭建的博客就建立了。我搭建它，并不是为了装b，而是人生中太多的事我们最终都会遗忘，记忆消逝了，那就什么也没有了。我希望用它来记录我的生活和成长，记录我平凡的一生。

心得

搭建博客并没有那么顺利，最近两天都在忙这个事情。中间出现了各种各样的问题，我没想着放弃，我热衷于自己喜欢的事情，好奇心也驱使着我探究下去。下面，我将会记录我搭建博客的每一步。

一、准备工作

到官网注册一个github账号

安装node.js

安装git for windows

二、博客建立

在你想要的位置里创建一个文件夹blog，然后右键选择Git Bash here

分别输入代码：

npm install hexo-cli -g

hexo init blog

cd blog

hexo server

三、写博客

在根目录下git bash here，输入：

hexo new filename

即可在source文件夹下自动生成_posts中的文章

四、常见命令

hexo new “postName” #新建文章

hexo new page “pageName” #新建页面

hexo generate #生成静态页面至public目录

hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）

hexo deploy #部署到GitHub

hexo help # 查看帮助

hexo version #查看Hexo的版本

缩写：

hexo n == hexo new

hexo g == hexo generate

hexo s == hexo server

hexo d == hexo deploy
